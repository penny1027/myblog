方案概述

Prometheus+Grafana是现代监控领域最流行的开源组合方案，提供强大的指标采集、存储、告警和可视化能力。

Prometheus

开源的监控系统和时间序列数据库，负责指标采集和存储

Grafana

领先的开源可视化平台，提供强大的数据展示和仪表盘功能

Exporters

各种采集器将系统/应用指标转换为Prometheus格式

部署步骤

1. Prometheus安装配置

# 下载Prometheus
wget https://github.com/prometheus/prometheus/releases/download/v2.30.3/prometheus-2.30.3.linux-amd64.tar.gz
tar xvfz prometheus-*.tar.gz
cd prometheus-*

# 配置prometheus.yml
global:
  scrape_interval: 15s

scrape_configs:
  - job_name: 'prometheus'
    static_configs:
      - targets: ['localhost:9090']

启动命令: ./prometheus --config.file=prometheus.yml

2. Grafana安装

# Ubuntu/Debian
sudo apt-get install -y adduser libfontconfig1
wget https://dl.grafana.com/oss/release/grafana_8.2.1_amd64.deb
sudo dpkg -i grafana_*.deb

# 启动服务
sudo systemctl daemon-reload
sudo systemctl start grafana-server
sudo systemctl enable grafana-server

访问地址: http://服务器IP:3000 (默认账号admin/admin)

3. Grafana配置数据源和仪表盘

  1. 登录Grafana后，左侧菜单选择"Configuration" → "Data Sources"
  2. 添加Prometheus数据源，URL填写http://prometheus-server:9090
  3. 导入官方仪表盘，ID如1860(Node Exporter)、7362(MySQL)等
  4. 配置告警规则和通知渠道(邮件、Slack等)

4. 常用Exporter部署

Node Exporter

wget https://github.com/prometheus/node_exporter/releases/download/v1.2.2/node_exporter-1.2.2.linux-amd64.tar.gz tar xvfz node_exporter-*.tar.gz cd node_exporter-* ./node_exporter

监控服务器基础指标

MySQL Exporter

wget https://github.com/prometheus/mysqld_exporter/releases/download/v0.13.0/mysqld_exporter-0.13.0.linux-amd64.tar.gz tar xvfz mysqld_exporter-*.tar.gz cd mysqld_exporter-* export DATA_SOURCE_NAME="user:password@(hostname:3306)/" ./mysqld_exporter

监控MySQL数据库指标

Blackbox Exporter

wget https://github.com/prometheus/blackbox_exporter/releases/download/v0.19.0/blackbox_exporter-0.19.0.linux-amd64.tar.gz tar xvfz blackbox_exporter-*.tar.gz cd blackbox_exporter-* ./blackbox_exporter

监控HTTP/HTTPS/TCP等服务可用性

最佳实践

架构设计建议

  • 生产环境建议Prometheus高可用部署(多实例+联邦集群)
  • 长期存储推荐Thanos或VictoriaMetrics方案
  • 按业务划分Prometheus采集任务(job)

监控策略建议

  • 黄金指标: 延迟、流量、错误、饱和度
  • 告警分级(P1-P4)并设置合理阈值
  • 重要业务指标配置SLO监控
